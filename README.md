LiveMTRX
========

Terminal "Matrix" style raining ASCII with random shifting colors and changing palettes. Runs on macOS; first target is terminal, later port to a live wallpaper.

Usage
-----

Run in terminal:

zsh run.sh

Project structure
-----------------

- src/: source code
- assets/: any assets
- run.sh: launcher
- build.sh: build script (optional)
- package.json: for tooling (optional)

Development
-----------

Designed for macOS using zsh. Requires Python 3.9+ or Node.js if you choose the JS version.
